# Internship Tracker

Full-stack internship tracker built with React + Vite on the frontend and Node.js + Express + MongoDB Atlas on the backend.

## Stack

- Frontend: React, Vite, React Router, Axios
- Backend: Express, Mongoose, JWT authentication
- Database: MongoDB Atlas

## Project structure

- `frontend/` React Vite client
- `backend/` Express API

## Quick start

1. Install dependencies:
   - `npm run install:all`
2. Copy env files:
   - `backend/.env.example` to `backend/.env`
   - `frontend/.env.example` to `frontend/.env`
3. Add your MongoDB Atlas connection string and JWT secret.
4. Run both apps:
   - `npm run dev`

## Default URLs

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000`
